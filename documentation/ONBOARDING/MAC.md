# Mac

Mac onboarding instructions. 

## Set Up Machine

Firstly, complete the initial setup of the machine.

### 1. Install OS Updates

Install whatever updates that are currently pending for your operating system.

### 2. Install Web Browser (Chrome, FireFox)

Install a browser through which you will access the internet. *Internet Explorer* is NOT RECOMMENDED.

### 3. Install SHELL Client (bash, zsh, etc.)

Install a sophisticated *SHELL* client for terminal command execution.

### 4. Install Package Manager (brew)

Install a *Package Manager* to standardize the set up of task execution environments.

## Install Software

With the initial setup complete, software can now be installed.
